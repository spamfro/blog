## Grid
Bootstrap uses 12 column grid
```
col-md-{n}  the n-th column of medium display size
col-xs-{n}  the n-th column of extra small display size
<div class="row"> <div class="col-xs-4"> ...4th column of a row ...</div> </div>
```
